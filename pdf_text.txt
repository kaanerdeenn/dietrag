●  Dil:  Python  3.x  ●  Orkestrasyon:  LangChain  &  LangGraph  ●  API  Sunucusu:  FastAPI  ●  Vektör  Veritabanı:  ChromaDB  ●  İlişkisel  Veritabanı:  SQLite  (Kullanıcı  profilleri  için)  ●  LLM  Modeli:  OpenAI  GPT-4o-mini  ●  Embedding:  OpenAI  Embeddings  ●  Web  Arama:  Tavily  Search  API
